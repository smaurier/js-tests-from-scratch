# 🧪 Étape 1 — Tests unitaires avec Jest

## 🎯 Objectif
Découvrir les bases des tests en JavaScript avec Jest.  
Apprendre à tester des **fonctions isolées** avec un cycle TDD (Red / Green / Refactor).

---

## 🔧 Ce qu’on a fait

- Écrit des fonctions simples :
  - `add(a, b)`
  - `capitalize(str)`
  - `truncate(str, maxLength)`
- Écrit leurs tests unitaires avec Jest

---

## 🧠 Concepts appris

| Concept           | Détail                                                                 |
|-------------------|------------------------------------------------------------------------|
| 🧪 Test unitaire    | Vérifie le résultat d’une **fonction unique**                         |
| 🔁 TDD             | Cycle : **Red (test échoue) → Green (code passe) → Refactor**         |
| ✍️ `test()`        | Déclare un test avec une description                                  |
| 📤 `expect().toBe()`| Vérifie un résultat précis (égalité stricte)                          |
| 💬 Cas limites     | Importance de tester des cas simples, vides, extrêmes, etc.           |

---

## 📦 Outils utilisés

- `jest`
- `pnpm` pour l’installation
- Tests exécutés en CLI via `pnpm test`

---

## ✅ Ce que tu sais faire maintenant

- Installer Jest
- Écrire des fonctions JavaScript testables
- Écrire un test unitaire simple et pertinent
- Comprendre la logique TDD
